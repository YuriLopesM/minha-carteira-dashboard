## 🍉 Installation

Create a `git clone` or download the source of the project. Use a package manager in order to install all dependencies. Examples:

```bash
 npm install
```

```bash
 yarn install
```

    
  
## 🌬️ Usage

To run this project, simple use:

```bash
 npm start
```

```bash
 yarn start
```
